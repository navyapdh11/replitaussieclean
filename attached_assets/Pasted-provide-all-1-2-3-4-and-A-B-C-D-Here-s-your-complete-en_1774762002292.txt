## 2️⃣ A. Xero + Stripe Integration Checklist (Enterprise-Grade)

### ✅ Pre-Integration Setup

| Step | Action | Details | Status |
|------|--------|---------|--------|
| 1 | Create Stripe Clearing Account in Xero | Account Type: **Bank (Clearing)**, Code: **Stripe Clearing** | ☐ |
| 2 | Enable GST on Income | Ensure all sales accounts have **GST on Income** enabled | ☐ |
| 3 | Create Expense Account for Fees | Account Name: **Stripe Fees**, Tax Code: **BAS Excluded** | ☐ |
| 4 | Connect Stripe to Xero | Settings → Payment Services → Add Stripe → OAuth flow | ☐ |
| 5 | Assign Stripe to Invoice Templates | Enable "Pay Now" button on all customer invoices | ☐ |

### ✅ Transaction Mapping (AU GST-Compliant)

| Stripe Transaction Type | Xero Account | GST Treatment | Notes |
|------------------------|--------------|---------------|-------|
| Customer Payment (Sale) | Sales Revenue | GST on Income | Code **full invoice amount** before fees |
| Stripe Processing Fee | Stripe Fees (Expense) | BAS Excluded | No GST to claim on fees |
| Customer Refund | Sales Revenue | GST on Income | Reduces revenue + GST liability |
| Payout to Bank | Stripe Clearing Account | N/A | Transfer only—no P&L impact |
| Dispute/Chargeback | Bad Debts Expense | GST on Income | Reverse original sale GST |

### ✅ Webhook Events to Subscribe To

Create these webhooks in **Stripe Dashboard → Developers → Webhooks**:

```json
{
  "url": "https://your-domain.com/api/webhooks/stripe",
  "events": [
    "payment_intent.succeeded",
    "payment_intent.payment_failed",
    "charge.refunded",
    "charge.dispute.created",
    "invoice.paid",
    "invoice.payment_failed"
  ]
}
```

### ✅ Bank Rules for Auto-Reconciliation

In Xero: **Accounting → Bank Accounts → Stripe Clearing → Create Bank Rule**

**Rule 1: Stripe Payout Inbound**
- Condition: `Description contains "STRIPE PAYOUT"`
- Transfer To: `Business Bank Account`
- No P&L impact

**Rule 2: Stripe Fee Deduction**
- Condition: `Description contains "STRIPE FEE"`
- Account: `Stripe Fees (Expense)`
- Tax Code: `BAS Excluded`

### ✅ Reconciliation Workflow (Weekly)

1. **Monday AM:** Match Stripe payouts to bank deposits
2. **Tuesday AM:** Review unmapped transactions in Stripe Clearing
3. **Wednesday:** Export Stripe Fee Report → Bulk-code to Stripe Fees account
4. **Friday:** Run BAS preview → Verify GST on Income matches Stripe sales

### ✅ Testing Checklist

| Test | Expected Result | Status |
|------|-----------------|--------|

| Issue $50 refund | Xero reverses $50 revenue + $5 GST, fee not refunded | ☐ |
| Simulate chargeback | Xero creates bad debt entry, reverses GST | ☐ |
| Run BAS report | GST on Income matches Stripe sales report | ☐ |
