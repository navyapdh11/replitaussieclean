### Updated Neo4j Seed Script (adds all 18 services + compliance edges)

**scripts/seed-all-services.cypher** (run once via Flux or kubectl exec)

```cypher
// Existing states + regulations remain
// NEW: 18 Service nodes
UNWIND [
  {id:"RES-DOM-001", name:"Weekly Domestic Cleaning", cat:"Residential"},
  {id:"RES-EOL-001", name:"End-of-Lease Bond Cleaning", cat:"Residential"},
  {id:"COMM-OFF-001", name:"Office Deep Clean", cat:"Commercial"},
  {id:"MED-HOSP-001", name:"Hospital-Grade Medical Cleaning", cat:"Medical"},
  // ... (full 18 – abbreviated here for brevity; full script in repo)
] AS svc
MERGE (s:Service {serviceId: svc.id, name: svc.name, category: svc.cat});

// Link to regulations
MATCH (s:Service), (reg:Regulation)
WHERE s.category = "Medical" AND reg.regId IN ["NHMRC-2019", "AS-NZS-4187"]
CREATE (s)-[:REQUIRES {mandatory: true}]->(reg);

MATCH (s:Service), (reg:Regulation)
WHERE s.category = "Residential" AND reg.regId = "ACL-2010"
CREATE (s)-[:REQUIRES {mandatory: true}]->(reg);
```
